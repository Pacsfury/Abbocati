# Abbocati
GAME IN EARLY DEVELOPMENT

---

## Game

Welcome to Abbocati! This game is an **Endless runner**, so in every round you will try to get as much points as possible. 

You are an avocado, which has been given the orders to go through The Kitchen, and other places. Be careful!, as there are planty of obstacles to be aware of.

## Obstacles

There are two (planned) obstacles:

| Name  | Description | 
| :---  | :---        |
| Knife | Defeats you |
| Stair | If you don't jump, you will crash |
| Kitchenbot | Sweeps the floor... and maybe you! |
